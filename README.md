# Word Count Tool

## Description
A Python utility that analyzes text files and counts the number of words, lines, and characters present in the file.

## Features
- Word counting
- Line counting
- Character counting
- Text analysis

## Technologies Used
- Python

## How to Run

```bash
python word_count.py
```

## Learning Outcomes
- String manipulation
- File handling
- Data analysis basics
